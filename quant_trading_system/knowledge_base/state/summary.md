# Long-horizon summary

This file is the rolling, multi-month memory for the harness. Each Claude that
runs daily is supposed to update this only when there's a real long-term
lesson worth carrying forward — not every day.

Examples of things that belong here:
  - "Mean-reversion strategies have outperformed trend-following in this
    paper account over the last quarter, particularly in low-VIX regimes."
  - "We've consistently been forced to widen stops on momentum strategies
    during earnings weeks; consider an earnings filter."
  - "The rolling-Sharpe threshold of 0.5 has rejected too many strategies
    during sideways months — it may be too tight."

Examples of things that do NOT belong here (those go in the daily conclusion
or, if relevant for tomorrow only, in the handoff):
  - Today's specific trades.
  - One-off market observations.
  - Reasoning for switching strategies (that's already in active_strategy.md).

This file is currently empty because the harness has not yet accumulated any
history.
